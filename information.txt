Informations générales :
	- L'accès au projet vous sera fourni via GitHub (master)
		https://github.com/DrakeXorn/CoffIG/tree/master

	- Pour pouvoir utiliser les fonctionnalités de notre application, vous devez installer le JDK 14 de Java.
		https://www.oracle.com/java/technologies/javase-jdk14-downloads.html
		
	- Pour que le programme se connecte à la base de données, vous devrez modifier le fichier dbConfig.properties se trouvant dans "resources". 
	Indiquez votre nom d'utilisateur après "user="
	Indiquez votre mot de passe après "password="
